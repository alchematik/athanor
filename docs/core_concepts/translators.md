# Translators

Translators translate configurations written in a programming language into a representation that Athanor understands.
The configuration that translators translate are both blueprints and provider schemas. 
This makes it so that users of Athanor never have to leave the comfort of their preferred programming language,
whether while creating resources with a blueprint or creating a new provider.
